Heart Disease Prediction

The Heart Disease Prediction System is a machine learning-powered web application designed to assess the likelihood of heart disease based on user-provided medical parameters such as age, blood pressure, cholesterol levels, heart rate, and other key health indicators. Built with HTML, CSS, and JavaScript for a user-friendly interface and powered by Python and Flask on the backend, the system processes inputs and applies a trained machine learning model to generate predictive insights. By leveraging data-driven algorithms, this project aims to raise awareness about heart health and assist users in identifying potential risks early. The integration of scikit-learn, Pandas, and NumPy ensures efficient data processing and model accuracy, making it a valuable tool in preventive healthcare.
